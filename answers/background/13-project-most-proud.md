# Describe a project you're most proud of.

**Target time:** 60 seconds (STAR)

---

## Talk track (virtualized-react — unique, shows depth)

> I'm most proud of **virtualized-react** — an open-source npm library I published for **rendering huge lists** in React smoothly.
>
> **Why it started:** At IQM and Atlys I kept hitting the same problem — tables and lists with hundreds or thousands of rows killing scroll performance. Existing libs were heavy or awkward for our use cases.
>
> **What I built:** Zero-dependency React virtualization — dynamic item heights, smart pre-fetching, TypeScript-first API. Works from 100 to 10,000+ items.
>
> **Why I'm proud:**
> - It solves a **real problem** I had in production, not a tutorial project
> - **Strangers use it** — npm downloads and community issues — that's accountability
> - It forced me to go deep on **rendering perf** — which directly helped my day job on Core Web Vitals
>
> I wasn't leading a team on it — it was **solo ownership** — design, implementation, docs, publishing, and responding to issues.
