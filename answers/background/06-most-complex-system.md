# What's the most complex system you've worked on?

**Target time:** 45–60 seconds

---

## Talk track (pick Atlys — strongest team + scale story)

> The most complex **production system at user scale** was **Atlys visa application workflows**.
>
> **Complexity came from:**
> - **High traffic** — thousands of users in multi-step onboarding
> - **Multi-step UI state** — forms, async validation, uploads
> - **Full-stack coupling** — React ↔ Fastify with strict contracts
> - **Performance pressure** — Core Web Vitals tied to conversion
>
> I wasn't a lead, but I **owned significant slices** — modular components (~30% faster rollouts), Fastify APIs, and perf work.
>
> **Boson** is complex in a different way — **Rust binary, SQLite encryption, YAML schema, embedded React build** — but it's **local tooling scale**, not thousands of concurrent consumer users. I'd lead with Atlys for "complex production system."
