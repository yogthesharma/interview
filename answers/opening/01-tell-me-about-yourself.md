**Target time:** 45–60 seconds  
**Format:** Spoken intro — don't recite resume line by line

---

## Talk track

> Hi, I'm Yog — I'm a Senior Product Engineer with about five and a half years of experience building production web systems, mostly with **React, TypeScript, and Node.js**.
>
> Most recently I've been building **Boson** — a local-first API client with a **Rust + SQLite** core and **React UI**, where API collections live as YAML in Git. Before that **Zyvia**, an AI-assisted task management app. Before that **Atlys**, where I owned high-traffic visa application workflows — React frontends, Fastify APIs, and Core Web Vitals performance work.

>
> What I enjoy most is **owning features from UI to runtime** — making them fast, reliable, and easy to ship. That's a big part of why this role interests me — building real platform software in a space that still needs better technology.
