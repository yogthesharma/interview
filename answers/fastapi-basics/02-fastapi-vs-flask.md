# FastAPI vs Flask — when do you pick each?

**Target time:** 30 seconds

---

## Talk track

> **FastAPI** — async APIs, schema validation, OpenAPI, new JSON services.
>
> **Flask** — WSGI ecosystem, Jinja SSR, legacy apps, team already on sync stack.
>
> **Migrate path:** new endpoints on FastAPI; Flask kept for admin until replaced.

---

## Avoid

- Rewriting working Flask solely for hype
