# JavaScript — conceptual (questions 70–97)

Runnable `.js` files with comments + edge cases for interview prep.

## How to use

```bash
node answers/javascript/01-event-loop.js
node answers/javascript/02-closures.js
# ... etc.
```

Read the **INTERVIEW ANSWER** blocks out loud, then run the file and trace output.
