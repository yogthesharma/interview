# What is S3 used for?

**Target time:** 45 seconds

---

## Talk track

> **S3** = object storage — files (PDF, CSV census, images), not database rows.
>
> **Common patterns:**
> - **Presigned URLs** — client uploads/downloads direct to S3; API never streams 50MB (api/14)  
> - **Static assets** — logos, generated PDFs  
> - **Lambda triggers** — `s3:ObjectCreated` → virus scan / parse census  
> - **Logs & backups** — CloudTrail, DB dumps  
> - **Lifecycle rules** — move to Glacier, expire after N days

> **Security:** private buckets, IAM policies, SSE encryption (SSE-S3 or KMS), block public access.

> **Example:** EOI documents, generated quote PDFs, employer census files — metadata in RDS, blob in S3.

## Code

```ts
// Presigned PUT — api/14
const uploadUrl = await getSignedUrl(s3, new PutObjectCommand({
  Bucket: "company-docs",
  Key: `acme/applications/${appId}/census.csv`,
  ContentType: "text/csv",
}), { expiresIn: 300 });
```
